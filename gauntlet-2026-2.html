The Rules

Keep It Fair, Keep It Moving

🔄

Player Rotation

  • Each event features a different team member
  • No single player can rep the team in every game
  • Everyone plays at least one event
🤝

Fair Play

  • No cheating, exploits, or outside help
  • Disconnects restart the segment if within the first 60 seconds
  • Disputes are resolved by majority vote
⏱️

Timing

  • 5-minute ready check before each game
  • Max 10-minute break between events
  • Tardiness forfeits that event's points
